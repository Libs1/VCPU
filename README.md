# VCPU

This virtual CPU is created in C language to perform registry level simulation of a CPU based on the ARM Cortex M0+ architecture. The CPUâ€™s memory size is 16Kbytes and uses 32-bit registers. The program has a simple user interface that would allow users to test machine level programs written for the CPU. The CPU executes various instructions based on the file in memory. The CPU has other features such as load file, memory dump, memory modify, write file, reset registers and display registers.
